## 递归

+  本质上,将原来的问题,转化成更小的同一问题

+  递归函数的调用,本质就是函数调用

+  只不过调用的函数是自己而已

### 注意

递归调用是有代价的:函数调用+ 系统栈空间

当数据量大时系统栈空间可能不够用
